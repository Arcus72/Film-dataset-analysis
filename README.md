# Film dataset analysis
Comprehensive Movie Dataset Analysis (700k+ records)

## Dataset
The dataset used in this analysis contains 701,103 entries, with each film described by 26 features. Further details regarding the dataset structure can be found in the 'Dataset structure' file.

Source: [Link](https://drive.google.com/drive/folders/1rk0FLNmMxsvSnKGb0aelmmrSmE7piS_X?usp=sharing)

## Files
- **Dataset Structure** – Detailed architecture and feature breakdown.
- **General Overview** – Comprehensive correlation and interdependency analysis.
- **Earning Analysis** (In Progress) – Analysis of film revenue drivers.
