# Film dataset analysis

Comprehensive Movie Dataset Analysis (700k+ records). This analysis has to goals:
- Prepare comprehensive analysis of every aspect of movie, actors, creators, directors and worlds.
- Identify specific features and trends that correlate with financial success in the film industry (if possible).
